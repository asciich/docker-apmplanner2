# APMPlanner2 container

Dockerized version of APMPlanner2 cround control software.

## Quickstart

```
git clone https://github.com/asciich/docker-apmplanner2.git
```

To use the currently latest version:

```
cd apmplanner_2_0_25
./run_container.sh
```
